Onmouseover Event

O evento onmouseover ocorre quando o ponteiro do mouse entra em um elemento. Esse evento é frequentemente usado junto com o onmouseout, que ocorre quando o ponteiro do mouse sai do elemento.

Onfocus Event

O onfocus ocorre quando um elemento recebe foco. O evento onfocus é frequentemente usado em campos de entrada.

Onchange Event

O evento onchange ocorre quando o valor de um elemento HTML é alterado.
Este evento é semelhante ao evento oninput. A diferença é que o evento oninput ocorre imediatamente após a alteração do valor de um elemento, enquanto onchange ocorre quando o elemento perde o foco, após a alteração do conteúdo. A outra diferença é que o evento onchange também funciona em elementos select.
